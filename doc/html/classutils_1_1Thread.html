void utils::Thread::sched_yield ( )
protected

Cette méthode libère le controle du thread et permet au processus appelant de traiter un autre thread de même priorité

Donne la main à un autre thread de meme priorité sched_yield() function checks to see if other processes at the same priority as that of the calling process are READY to run. If so, the calling process yields to them and places itself at the end of the READY process queue. The sched_yield() function never yields to a lower priority process.
