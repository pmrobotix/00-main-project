void logs::end ( logs::Logger::LoggerBuffer &  buffer)

Cette méthode permet de finir les flux de construction par la notation.

<< logs::end;

.

Parameters
bufferLe buffer de référence

L'utilisation de cette méthode repose sur les concepts avancés de C++ et en particulier le fait que lors de la compilation cette méthode sera effectivement appelé comme si le code était ;

logs::end(logger.info() << "mon message");

Bien que ce concept puisse paraitre étrange, c'est sur cette même fonctionnalité que std::endl est basée.
