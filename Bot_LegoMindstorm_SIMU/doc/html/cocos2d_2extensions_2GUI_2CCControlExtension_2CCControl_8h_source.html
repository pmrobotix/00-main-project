53 /*
54  * @class
55  * Control is inspired by the UIControl API class from the UIKit library of
56  * CocoaTouch. It provides a base class for control Sprites such as Button
57  * or Slider that convey user intent to the application.
58  *
59  * The goal of Control is to define an interface and base implementation for
60  * preparing action messages and initially dispatching them to their targets when
61  * certain events occur.
62  *
63  * To use the Control you have to subclass it.
64  */
