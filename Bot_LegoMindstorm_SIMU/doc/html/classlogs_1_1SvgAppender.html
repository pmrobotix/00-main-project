Detailed Description

Implémentation de Appender créant un fichier SVG.

Les messages envoyés à cet appender sont considérés comme des éléments SVG.

void logs::SvgAppender::flush ( )
virtual

Cette méthode clos le fichier de référence.

Cette méthode est appelée automatiquement par le destructeur de la classe mais peut être exploitée avant.

Une fois cette méthode appelée aucun nouveau message ne doit être envoyé a cet Appender.
