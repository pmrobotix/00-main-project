Detailed Description

template<class K, class V>
class Map< K, V >

Similar to std::unordered_map, but it will manage reference count automatically internally. Which means it will invoke Ref::retain() when adding an element, and invoke Ref::release() when removing an element.

Warning
The element should be Ref or its sub-class. NA NA
