void Device::vibrate ( float  duration)
static

Only works on iOS devices that support vibration (such as iPhone). Should only be used for important alerts. Use risks rejection in iTunes Store.

Vibrate for the specified amount of time. If vibrate is not supported, then invoking this method has no effect. Some platforms limit to a maximum duration of 5 seconds. Duration is ignored on iOS due to API limitations.

durationThe duration in seconds.
