25 *using spec version 1 found at https://github.com/LearnBoost/socket.io-spec

27 Usage is described below, a full working example can be found in TestCpp under ExtionsTest/NetworkTest/SocketIOTest
28 
29 creating a new connection to a socket.io server running at localhost:3000
30 
31  SIOClient *client = SocketIO::connect(*delegate, "ws://localhost:3000");
32 
33 the connection process will begin and if successful delegate::onOpen will be called
34 if the connection process results in an error, delegate::onError will be called with the err msg
35 
36 sending a message to the server
37 
38  client->send("Hello!");
39 
40 emitting an event to be handled by the server, argument json formatting is up to you
41 
42  client->emit("eventname", "[{\"arg\":\"value\"}]");
43 
44 registering an event callback, target should be a member function in a subclass of SIODelegate
45 CC_CALLBACK_2 is used to wrap the callback with std::bind and store as an SIOEvent
46 
47  client->on("eventname", CC_CALLBACK_2(TargetClass::targetfunc, *targetclass_instance));
48 
49 event target function should match this pattern, *this pointer will be made available
50 
51  void TargetClass::targetfunc(SIOClient *, const std::string&)
52 
53 disconnect from the endpoint by calling disconnect(), onClose will be called on the delegate once complete
54 in the onClose method the pointer should be set to NULL or used to connect to a new endpoint
55 
56  client->disconnect();
