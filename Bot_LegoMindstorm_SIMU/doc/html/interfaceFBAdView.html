Detailed Description

A customized UIView to represent a Facebook ad (a.k.a. banner ad).

- (instancetype) initWithPlacementID: (NSString *)  placementID
adSize: (FBAdSize)  adSize
rootViewController: (UIViewController *)  viewController 

This is a method to initialize an FBAdView matching the given placement id.

Parameters
placementIDThe id of the ad placement. You can create your placement id from Facebook developers page.
adSizeThe size of the ad; for example, kFBAdSizeHeight50Banner or kFBAdSizeHeight90Banner.
rootViewControllerThe view controller that will be used to present the ad and the app store view.
