53 /* The DynamicAttribute class or its child classes encapsulate an attribute with specific (dynamic) behaviour.
54 @remarks
55  This class provides a uniform interface for retrieving the value of an attribute, while the calculation of
56  this value may vary. Each subclass provides its own implementation of the getValue() function and the calling
57  application doesn't need to know about the underlying logic. A subclass could just return a value that has
58  previously been set, but it can also return a value that is randomly generated by a pre-defined min/max interval.
59  The DynamicAttribute class is used in situations where different behaviour of a certain attribute is needed,
60  but where implementation of this behaviour may not be scattered or duplicated within the application that needs
61  it.
62 */

65  public:
66  enum DynamicAttributeType
67  {
68  DAT_FIXED,

72  };
