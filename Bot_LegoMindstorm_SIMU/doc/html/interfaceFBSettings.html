◆ enablePlatformCompatibility:() [1/5]

+ (void) enablePlatformCompatibility: (BOOL)  enable

Configures the SDK to use the legacy platform.

Parameters
enableindicates whether to use the legacy mode Setting this flag has several effects:
  • FBRequests will target v1.0 of the Graph API.
  • Login will use the prior behavior without abilities to decline permission.
  • Specific new features such as FBLikeButton that require the current platform will not work.
