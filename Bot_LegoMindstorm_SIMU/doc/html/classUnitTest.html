Detailed Description

Cette classe permet de créer des tests unitaires.

Chaque classe de test devra hériter de cette classe et définir la méthode UnitTest::suite() qui lancera les différents tests unitaires associés.

Tous les tests doivent avoir un nom propre définit à la construction de l'objet.

Les méthodes UnitTest::assert() et UnitTest::fail() permettent de valider les tests effectués. La première vérifie qu'un test booléen est vrai, la seconde arrète le test en cours et retourne une erreur.

Les classes héritant de la classe UnitTest devrait respecter les points suivants :

  • Le nom de la classe se compose du nom de la classe testée, suffixé par Test.
  • Les méthodes de tests sont préfixées par test.
  • Toutes les méthodes de tests sont appelées par la méthode suite().
